# Progress Program v0.3.0 개발 계획서

**개발 상태**: 🚧 **개발 중 (In Development)**

---

## 📋 **개요**

v0.3.0은 "간단함을 유지하면서 실용성 극대화"를 목표로 하며, 3개 핵심 기능만을 신중히 선택하여 구현합니다:

1. **키보드 단축키**: 작업 효율성 향상
2. **백업/복원**: 데이터 안전성 확보  
3. **마감일 표시**: 기본적인 일정 관리

---

## 🗓️ **개발 일정**

### Phase 1: 키보드 단축키 (1주)
- [ ] 메뉴바에 단축키 표시 추가
- [ ] QShortcut 기반 이벤트 처리
- [ ] 컨텍스트 감지 로직 구현

### Phase 2: 백업/복원 (1주)  
- [ ] 파일 메뉴에 백업/복원 옵션 추가
- [ ] 데이터베이스 파일 복사 및 검증
- [ ] 사용자 피드백 시스템

### Phase 3: 마감일 관리 (1주)
- [ ] Task 모델에 due_date 필드 추가
- [ ] 마감일 설정 UI 구현
- [ ] 시각적 상태 표시 시스템

### Phase 4: 통합 및 테스트 (3일)
- [ ] 전체 기능 통합 테스트
- [ ] 버그 수정 및 최적화
- [ ] 문서화 완성
